MyFlix API Dokumentation

Übersicht

Die MyFlix API ermöglicht den Zugriff auf Filmdaten aus einer Datenbank. Sie können Informationen über Filme abrufen, einschließlich Titel, Beschreibung, Genre und mehr.

Endpunkte

Filme abrufen

GET /api/filme

Diese Anfrage gibt eine Liste aller verfügbaren Filme zurück.

Einzelnen Film abrufen

GET /api/filme/{id}

Diese Anfrage gibt Informationen zu einem bestimmten Film zurück, basierend auf der angegebenen ID.

Film erstellen

POST /api/filme

Diese Anfrage erstellt einen neuen Film in der Datenbank.

Film aktualisieren

PUT /api/filme/{id}

Diese Anfrage aktualisiert die Informationen eines bestimmten Films, basierend auf der angegebenen ID.

Film löschen

DELETE /api/filme/{id}

Diese Anfrage löscht einen bestimmten Film aus der Datenbank, basierend auf der angegebenen ID.
